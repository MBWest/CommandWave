Command Wave

Command Wave is a web-based interface featuring integrated multi-tab terminals (powered by ttyd and tmux). It allows users to load Markdown playbooks, substitute variables dynamically per tab, and execute commands directly within the embedded terminals. The application also includes persistent global and tab-specific notes for efficient workflow management.

View on GitHub

Key Features

  • Integrated multi-tab terminals (ttyd + tmux)
  • Load and display Markdown (.md) playbooks
  • Execute code blocks from playbooks directly into the active terminal
  • Per-tab variable substitution (e.g., $TargetIP)
  • Persistent floating notes panels (Global and Tab-specific)
  • Web-based UI with a cyberpunk/neon aesthetic
Important Note: This is a static information page for the Command Wave project. The actual application is built using Python and Flask and requires a backend server to run. It cannot be run directly from GitHub Pages. Please refer to the main repository for setup and usage instructions to run the application locally.

Getting Started

To use Command Wave, you need to clone the repository and run the application locally on your machine. Please ensure you have the prerequisites (Python 3, pip, ttyd, tmux) installed.
